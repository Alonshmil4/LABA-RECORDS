# אשכולות מילות מפתח — LABA RECORDS

מסמך עזר לסוכן SEO. לא לערוך את האתר ישירות מכאן.

## עדיפות 1 — מקומי + שירות ליבה

| חיפוש | כוונה | הערות |
|--------|--------|--------|
| אולפן הקלטות בשרון | מקומי | מיקום אמיתי: מושב אודים |
| אולפן הקלטות במרכז | אזור שירות | נגישות מת"א / מרכז, לא כתובת |
| אולפן הקלטות בנתניה | אזור שירות | מרחק / נגישות, בלי טענת כתובת בנתניה |
| אולפן הקלטות | כללי | מותג + אזור בגוף העמוד |
| הקלטת שיר | שירות | FAQ, תהליך, hero |

## עדיפות 2 — שירותים וסמנטיקה

הפקה מוזיקלית, עריכה ומיקס, מיקס ומאסטר, הקלטת ווקאל, הקלטת גיטרה, אולפן אינדי, אולפן בין העצים, סשן באולפן, לקביעת סשן.

## עדיפות 3 — מותג ואמון

LABA RECORDS, קורל ביסמוט, אולפן קורל ביסמוט, הפקות קורל ביסמוט.

## עדיפות 4 — שאלות (FAQ / long-tail)

כמה עולה הקלטת שיר, כמה זמן לוקח להקליט שיר, מה לקחת לסשן הקלטה, איך מתכוננים להקלטה, יש חניה באולפן, איפה האולפן.

## מיפוי לעמוד (one-pager)

| אזור | מילות מפתח טבעיות |
|------|-------------------|
| `<title>` / meta | אולפן הקלטות בשרון + מותג |
| meta description | שירותים + מרכז/נתניה כנגישות |
| JSON-LD `areaServed` | השרון, מרכז, נתניה, תל אביב |
| Hero / about | אולפן הקלטות, מושב אודים, השרון |
| תהליך / FAQ | הקלטת שיר, הכנה, מחיר, חניה |
| גלריה / alt | אולפן, חלל, סשן — בלי רשימת מילים |

## כללי שימוש

- מילת מפתח ראשית פעם אחת בכותרת/מטא; שאר הפעמים בגוף בטבעיות.
- לא לשכפל אותה משפט בשלושה מקומות סמוכים.
- עברית RTL; אל תתרגם מונחים מותגיים לאנגלית בטקסט גלוי אם האתר בעברית.
