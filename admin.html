CSS גלובלי (ייכנס ל-styles.css)

כפתור «עדכון (לייב לקוד)» כותב מיד ל־site-content.json ומריץ Apply על index.html/styles.css. אם תרצו גם קובץ גיבוי, השתמשו ב"הורדה". ידני: שמרו את הקובץ בשורש הפרויקט והריצו npm run admin:apply — זה יעדכן את index.html ואת בלוק ה-CSS ב־styles.css.
