# LABA RECORDS (דמו ראשוני)

אתר סטטי ראשוני בעברית (RTL) עבור **LABA RECORDS** — קורל ביסמוט.

## מה יש כאן

- `index.html`: דף One‑pager עם הסקשנים מהמצגת
- `styles.css`: עיצוב כהה/קולנועי בהשראת האתרים ששלחת
- `main.js`: תפריט מובייל, FAQ אקורדיון, ומודאל “לקביעת סשן” (דמו)

## איך מריצים

פתח/י את `index.html` בדפדפן (דאבל‑קליק).

אם תרצה/י “לוקאל סרבר” (מומלץ כדי למנוע בעיות טעינה בדפדפנים מסוימים), אפשר ב‑PowerShell מתוך התיקייה:

```bash
python -m http.server 5500
```

אם אין אצלך Python מותקן, תגיד/י לי ונריץ עם כלי אחר (Node / Live Server).

- וידאו רקע/תמונות בגלריה (כרגע placeholders)
- לינקים אמיתיים להפקות (Spotify/YouTube)
- לינקים לרשתות (Instagram וכו׳)
